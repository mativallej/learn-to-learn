**Concepto**  
El cuaderno físico funciona como laboratorio portátil: escribir a mano aclara el
pensamiento, obliga a sintetizar y genera ideas que no aparecerían solo hablando.

**Setup**  
- Cuaderno espiral con tapas rígidas que permiten arrancar páginas y trabajar
  plano.  
- Tamaño bolsillo para llevarlo siempre.  
- Papel agradable; si no provoca escribir, cámbialo.  
- Plumas preferidas: Uniball micro 0.5 y Muji azul oscuro.

**Proceso**  
- Escribe mucho y arranca páginas para poder ver varias ideas a la vez.  
- Considera al cuaderno desechable: dura ~2–3 semanas antes de migrar lo útil a
  tu sistema digital.  
- Usa la escritura como herramienta de pensamiento, no como archivo pasivo.  
- Cuando terminas con una idea, destruye la página o archívala en fotos si hace
  falta.
