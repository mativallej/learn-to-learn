**Concepto**  
Zettelkasten (caja de notas) es un sistema para pensar escribiendo: cada nota atomica vive sola pero se enlaza con otras mediante referencias y contexto minimo.

**Origen**  
Creado por Niklas Luhmann para investigar sociologia con miles de fichas numeradas. Adaptado hoy a herramientas como Obsidian, Notion o papel.

**Componentes basicos**  
- **Notas atomicas**: idea unica + referencia.  
- **Indices o mapas**: notas puente que agrupan rutas tematicas.  
- **Enlaces**: relaciones bidireccionales (`[[Nota X]] ↔ [[Nota Y]]`).

**Flujo sugerido**  
1. Captura una idea cruda en tu inbox diario.  
2. Destila la idea en una nota atomica usando la plantilla anterior.  
3. Enlaza la nota con minimo dos nodos existentes (ej. `[[Funnel]]`, `[[Behavioral Economics]]`).  
4. Actualiza un mapa o dashboard para descubrir huecos.

**Beneficios**  
- Pensamiento compuesto: cada enlace crea nuevas combinaciones.  
- Preparacion para creacion de contenido (videos, articulos) sin partir de cero.  
- Memoria externa que sirve para revisitar aprendizajes Lean Startup y Producto.

**Buenas practicas**  
- Escribe con tus palabras; cita fuentes solo cuando importe.  
- Añade fecha o contexto (proyecto, problema) si ayuda a rastrear decisiones.  
- Mantén consistencia en tags (`#product`, `#behavioral`) para buscar rapido con `rg '#product'`.  
- Revisa semanalmente notas huérfanas y enlazalas o elimínalas si no aportan valor.
